# Roomy

> **Find your perfect roommate before move-in.**

Roomy is a roommate discovery platform built exclusively for **Shiv Nadar University** students. It helps incoming and current students connect with compatible roommates based on their lifestyle, preferences, and living habits—making hostel allocation less random and much more comfortable.

**Live:** https://roomy-snu.vercel.app

---

## Features

* Exclusive to **Shiv Nadar University** students
* Create a personalized roommate profile
* Browse potential roommates
* Express interest in students you'd like to connect with
* Share your lifestyle preferences and room expectations
* Fully responsive across desktop and mobile devices

---

## Running Locally

Clone the repository:

```bash
git clone <repository-url>
cd roomy
```

Install dependencies:

```bash
npm install
```

Create a `.env.local` file with the required environment variables.

Start the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

## Vision

Finding a roommate shouldn't rely on random assignments or scattered WhatsApp groups.

Roomy aims to help SNU students find roommates they're genuinely compatible with before moving into their hostel, making campus life more comfortable from day one.
